# 第三章：CSS基礎

## 3.1 CSS簡介

CSS（層疊樣式表）是一種用於描述網頁的外觀和樣式的語言。它與HTML一起使用，用於控制網頁元素的佈局、字型、顏色、背景、邊框和其他視覺效果。

CSS的主要目標是將內容和表現分離。這意味著可以將HTML文件中的內容與CSS文件中的樣式定義分開，使得網頁設計更具靈活性和可維護性。通過使用CSS，可以在整個網站中輕鬆地應用一致的樣式，同時提供更好的可讀性和維護性。

## 3.2 CSS選擇器和屬性

# 3.2.1 CSS選擇器

CSS選擇器和屬性是用於選擇HTML元素並為其應用樣式的關鍵部分。選擇器用於選擇元素，而屬性則用於定義元素的樣式，以下是一些常見的CSS選擇器：

元素選擇器：選擇特定的HTML元素，例如p選擇所有段落元素。

```html
p {
  /* 樣式定義 */
}
```

類別選擇器：選擇具有特定類別屬性的元素，例如.highlight選擇所有具有highlight類別的元素。

```html
.highlight {
  /* 樣式定義 */
}
```

ID選擇器：選擇具有特定ID屬性的元素，例如#header選擇具有header ID的元素。

```html
#header {
  /* 樣式定義 */
}
```

屬性選擇器：選擇具有特定屬性值的元素，例如[type="text"]選擇所有type屬性值為"text"的元素。

```html
[type="text"] {
  /* 樣式定義 */
}
```

群組選擇器：選擇多個元素，並將相同的樣式應用於它們。

```html
h1, h2, h3 {
  /* 樣式定義 */
}
```

除了這些基本選擇器，還有許多其他選擇器，如子選擇器、後代選擇器、相鄰兄弟選擇器等，可以根據需要使用。

### 3.2.2 CSS屬性

CSS屬性用於定義元素的樣式。以下是一些常見的CSS屬性：
color：定義文字顏色。

```html
p {
  color: red;
}
```
font-size：定義文字大小。

```html
p {
  font-size: 16px;
}
```
background-color：定義背景顏色。


```html
div {
  background-color: #f0f0f0;
}
```
width和height：定義元素的寬度和高度。

```html
img {
  width: 200px;
  height: 150px;
}
```
margin和padding：定義元素的外邊距和內邊距。

```html
div {
  margin: 10px;
  padding: 20px;
}
```
border：定義元素的邊框。

```html
div {
  border: 1px solid black;
}
```
text-align：定義文字的對齊方式。

```html
p {
  text-align: center;
}
```
display：定義元素的顯示方式。

```html
span {
  display: block;
}
```
position：定義元素的定位方式。

```html
div {
  position: absolute;
  top: 0;
  left: 0;
}
```
這只是一些常見的CSS屬性示例，CSS提供了許多其他屬性，用於定義元素的外觀和行為。通過選擇合適的屬性並為其指定適當的值，您可以創建出各種各樣的網頁設計效果。

瞭解和掌握CSS選擇器和屬性對於設計和開發具有吸引力和交互性的網頁至關重要。它們使您能夠以獨特的方式定制和美化HTML元素，為網站創造出令人印象深刻的外觀和用戶體驗。

## 3.3 CSS盒模型

CSS盒模型是一種佈局模型，用於描述HTML元素在頁面上的佈局和定位。它將每個元素看作一個矩形的盒子，並定義了該盒子的屬性和行為。

CSS盒模型由以下四個部分組成：

內容框（Content Box）：這是盒子的內容區域，包含文本、圖像或其他元素的實際內容。

邊框（Border）：位於內容框周圍的線條，用於分隔元素的內容和周圍的區域。

內邊距（Padding）：內邊距是內容框與邊框之間的空間，可以在內容和邊框之間添加填充。

外邊距（Margin）：外邊距是盒子周圍的空間，用於控制元素與其他元素之間的間距。

這些部分的組合形成了完整的盒子模型，並通過屬性進行控制和調整。每個部分都有相關的CSS屬性，用於定義其大小、顏色、邊距等屬性。

例如，以下是一個使用CSS盒模型的示例：

```html
div {
  width: 200px; /* 盒子的寬度 */
  height: 150px; /* 盒子的高度 */
  border: 1px solid black; /* 邊框的寬度和顏色 */
  padding: 20px; /* 內邊距 */
  margin: 10px; /* 外邊距 */
}
```
上述示例中，div元素被定義為一個具有特定寬度、高度、邊框、內邊距和外邊距的盒子。根據這些屬性的設置，該元素將在頁面上佔據一個矩形區域。

通過理解和運用CSS盒模型，您可以更好地控制和設計網頁的佈局，創建出各種各樣的排版效果。了解如何使用內容框、邊框、內邊距和外邊距等屬性，可以幫助您在網頁設計中實現所需的外觀和間距效果。

## 3.4 CSS布局

CSS布局是指在網頁中對元素進行定位和排列的過程。它涉及到如何控制元素的位置、大小、對齊和相對關係，以實現所需的網頁結構和外觀。

在CSS中，有多種方法可以實現不同類型的布局。以下是一些常用的CSS布局技術：

流式布局（Flow Layout）：流式布局是最基本的布局方式，元素按照它們在HTML中出現的順序自動佈局。元素根據可用空間的大小自動調整位置，並自動換行。

浮動布局（Float Layout）：浮動布局通過將元素浮動到左或右側，使其脫離正常的文檔流。浮動元素可以實現多欄佈局，並且可以對元素進行自由的位置調整。

定位布局（Positioning Layout）：定位布局使用CSS的定位屬性（如position: absolute和position: relative）來控制元素的精確位置。這種布局方式常用於創建特定的視覺效果或實現元素的重疊和層次關係。

網格布局（Grid Layout）：網格布局是一種二維佈局系統，可以將元素按照網格的行和列進行排列。通過定義網格容器和網格項目的屬性，可以實現複雜的網頁佈局。

彈性盒子布局（Flexbox Layout）：彈性盒子布局是一種用於排列元素的彈性模型，它基於主軸和交叉軸的概念，可以靈活地調整元素的大小和位置。

以上僅是一些常見的CSS布局技術，每種技術都有其特定的使用場景和適用性。根據網頁的需求和設計要求，您可以選擇適合的布局技術或組合多種技術來實現所需的佈局效果。

理解和掌握CSS布局技術對於創建具有良好結構和外觀的網頁至關重要。通過適當的布局，您可以實現元素的合理排列和對齊，提升用戶體驗和訪問性，並確確保網頁的內容清晰可讀。此外，使用適當的布局技術還可以實現響應式設計，使網頁能夠適應不同螢幕大小和設備類型。

透過學習和實踐，您可以逐漸熟悉不同的CSS布局技術，並能夠靈活運用它們來創建多樣化且具有吸引力的網頁佈局。請記住，熟練掌握CSS布局需要時間和經驗，持續學習和實踐是提高技能的關鍵。

## 3.5 CSS文本和字體樣式

在網站設計中，CSS提供了豐富的文本和字體樣式屬性，可以用於控制網頁中的文本內容的外觀和排版。以下是一些常用的CSS文本和字體樣式屬性：

字體大小（font-size）：用於設定文本的字體大小，可以使用絕對單位（如像素）或相對單位（如百分比）。

字體類型（font-family）：用於指定文本所使用的字體，可以使用具體的字體名稱，或使用通用字體類型（如sans-serif、serif、monospace）。

字體粗細（font-weight）：用於設定文本的字體粗細，可以使用數值（如400、700）或關鍵字（如normal、bold）。

字體斜體（font-style）：用於設定文本的字體樣式，可以設定為斜體（italic）或正常（normal）。

文本顏色（color）：用於設定文本的顏色，可以使用具體的顏色值（如十六進制、RGB值）或預定義的顏色名稱。

文本對齊（text-align）：用於設定文本的對齊方式，可以設定為左對齊（left）、右對齊（right）、居中對齊（center）或兩端對齊（justify）。

行高（line-height）：用於設定文本行高，控制文本行之間的垂直間距。

文字修飾（text-decoration）：用於添加文字修飾效果，如下劃線（underline）、刪除線（line-through）等。

文本間距（letter-spacing）：用於設定字母之間的間距。

文字陰影（text-shadow）：用於添加文字陰影效果，可以設定陰影的顏色、位置和模糊度。

這些CSS屬性可以單獨應用於特定的文本元素，也可以通過選擇器選擇多個元素進行統一的樣式設定。通過適當的使用和組合，您可以創建出各種吸引人的文本和字體樣式，提升網頁的可讀性和視覺效果。

請注意，不同瀏覽器可能對CSS文本和字體樣式的支援程度有所不同，因此在設計網站時，建議測試不同瀏覽器上的顯示效果，並做必要的調整和優化。

除了上述提到的屬性，CSS還提供了其他一些用於控制文本和字體樣式的屬性，如文本轉換（text-transform）用於轉換文本的大小寫形式（如大寫、小寫、首字母大寫等），以及文本溢出（text-overflow）用於處理文本內容超出容器範圍的情況。

此外，您還可以通過使用CSS框架（如Bootstrap、Foundation等）來快速應用預定義的文本和字體樣式，這些框架提供了一系列的CSS類和樣式，可用於快速建立具有一致風格的網頁。

在設計網站時，適當地運用CSS文本和字體樣式可以提升用戶體驗，優化內容呈現，並創造出令人印象深刻的網頁設計。建議熟悉不同的CSS文本和字體樣式屬性，並根據項目需求和設計目標來選擇和應用適當的屬性，以實現所需的視覺效果。
